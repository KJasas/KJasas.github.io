Data scraping

Briefing

Let's say we have a lot (e.g. thousands) of different url sources, which need to to be clasified (e.g. government, news outlets, etc.). Normally, a person should visit every site, gather basic information about the source and put the source in a category.

The code I've written takes every url, cuts it to the base site, visits it and saves the information which is between its html code Title *** /Title parts.

For example, source http://www.platts.com/news-feature/2015/metals/non-ferrous-metals-outlook/alumina-asia would become http://www.platts.com/ and would return “Latest Oil, Energy & Metals News, Market Data and Analysis | Platts”.
